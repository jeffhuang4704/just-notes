### Bootstrap modal with bigger size

```HTML

// BootStrap modal, make it bigger via "modal-large"
<div class="modal fade" id="modal_preview_pdf" tabindex="-1" role="dialog" aria-labelledby="modalPreviewPDF" aria-hidden="true">
    <div class="modal-dialog modal-large" role="document">


//CSS
.modal-large {
    max-width: 900px;
    width: 900px;
}


```
